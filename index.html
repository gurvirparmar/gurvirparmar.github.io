Projects

A GUI based Book Store Application implemented using State Design Pattern in Java/JavaFX. Consisted of the 'owner' of the book store being able to add and remove books in the inventory, and any registered 'customers' in the database who shopped at the store looking to purchase some books. Customers were rewarded with points depending on the amount they spent purchasing these books and had the ability to redeem these points. All information was stored into a database file which contained all this information and was updated instantaneously.

Designed and Implemented a Multicast Overlay Network using socket APIs and distributed routing algorithms in Python. Built a multicast distribution tree at each node and maintained specified packet flow. TCP socket was used to establish connectivity with peers for exchanging messages. Nodes in the network read the configuration file during the bootstrap process, and a multicast routing table opened connections with peer nodes.

Created and implemented own network Secure Socket Layer (SSL) Protocol from scratch using Java. This consisted of creating the handshake protocol and the exchange of data and certificates between the server and client. For the creation of the keys, RSA cipher was used for the creations of the asymmetric keys and the DES cipher for the creation of the symmetric key. With the public and private key pair, a self-signed certificate was generated where, when sent to another device, it was guaranteed that the public key that is inside the certificate was issued by the device that signed the certificate.

Designed and implemented an embedded system suitable for a proposed embedded media center application. The system consisted of an ARM-Cortex M3 processor on a MCB1700 Dev Board with serial interface and parallel interfaces for LCD, joysticks, speakers, etc... It was then configured and implemented to run this real-time application based on an RTOS of RTX using Keil uVision.
